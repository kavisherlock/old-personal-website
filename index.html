Work

My work at my job primarily deals with front-end development, from C++ and C# Windows Desktop applications to web applications using Ruby on Rails and ReactJS.

On my own, I also like to work with machine learning algorithms. Constantly reading papers and trying to implement new algorithms, I'm trying to keep up to the ever-growing world of machine learning.

In Progress

StreetViewPlayground

Neural Networks for the solving the Street View House Numbers dataset. This is a prequel to a text detection mobile application I intend. The dream is to make a very basic clone of Google Lens.

SpotifyClient-React

A Spotify client using ReactJS and Redux. Uses the Spotify API to retrieve a user's playlists and tracks.

SmartChess

Implementing different algorithms for an AI/ML-powered chess engine. The goal is to make a robot arm that can play chess in real time. Starting with a software version before I work on the hardware.

ReactAnimationPlayground

Using various animation libraries to learn more about animating in React. Currently use CSS, ReactTransitionGroup, React Motion, React Spring, React Pose, Velocity React, etc.

Completed/On Hold

NeuralNetworkPlayground

Various ways to implement Neural Networks (and other machine learning techniques). All are trained, evaluated and tested using the MNIST digit recognition dataset. I used it primarily as a way to introduce myself to neural networks and the many tools available

BlackJack-RL

Various Reinforcement learning algorithms for multi- and single-player BlackJack I used it primarily as a way to introduce myself to reinforcement learning and its many algorithms

GithubClient-ReactNative

A GitHub client using React Native. I used it as my introduction to React Native and as part of the React Native introductory presentation I gave. Uses the GitHub API to retrieve user information.

TwitterClone-RoR

A clone of Twitter using Ruby on Rails. It has all the basic features of Twitter (Tweeting, Profiles, Following). It was my first Ruby of Rails project. In the future, I'd like to continue adding more complicated features.

Eight-Puzzle-Solver

An 8-puzzle solver using various AI search algorithms. First Visual Studio project from scratch. Two types of algorithms were used: uninformed search and informed search with various heuristics
